# Effortless Charts :

### 🌐[Click Here to Visit Website](https://effortless-charts.web.app)

- Effortless Chart is a Quick and Easy Chart Maker Website with minimal Input.
- Just Add the data you need and the chart title and type and in a moment you get presented with a colorful chart that can be downloaded and used Anywhere.
